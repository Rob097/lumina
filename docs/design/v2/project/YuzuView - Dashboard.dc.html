Install on any websiteThree steps. Copy, paste, verify.
1Add the script
Paste this just before </head>. Your publishable key is public by design.
HTML
<script src="https://cdn.yuzuview.com/v1.js" data-key="pk_live_8f2a9c4e"></script>
2Place the button
Drop this where you want the launcher, or call window.Lumina.open(...) from your own button.
HTML
<div data-lumina-product="SKU-HALO-01"></div>
3Verify it’s live
Click the button — the modal appears.
Allow-list your domain in Settings → Domains, or requests are blocked.
Watch runs appear in Generations and metrics in Overview.
